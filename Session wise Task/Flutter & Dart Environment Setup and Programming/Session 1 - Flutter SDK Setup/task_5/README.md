# Insta Clone

This is a Flutter-based Instagram clone project.

## Project Structure

Below are the main folders used in this Flutter project and their specific roles:

- **lib/**: This is the heart of the project where all the Dart source code lives. It contains the UI components, business logic, and platform-independent code.
- **android/**: This directory contains the source code and configuration for the Android version of the app. It is used for Android-specific settings, icons, and platform integrations.
- **ios/**: This folder holds the Xcode project and configuration for the iOS version of the app. It manages Apple-specific settings, certificates, and app store configurations.
- **test/**: This directory is dedicated to automated testing. It contains unit tests and widget tests to ensure the application behaves correctly and remains stable during development.
