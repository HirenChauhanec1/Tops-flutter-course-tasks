Launching lib\main.dart on Redmi Note 13 pro plus in debug mode...

Try 'adb kill-server' if that seems wrong.
Otherwise check for a confirmation dialog on your device.


AI Suggested Steps:

Disconnect and reconnect the Android device.
Enable Developer Options and USB Debugging on the phone.
Accept the "Allow USB debugging" prompt on the device.
Restart the ADB server:
      1.adb kill-server
      2.adb start-server
      3.adb devices
Run the Flutter app again:
      flutter run


Summary :

The issue was caused because my Android device was not authorized for USB debugging.
After accepting the USB debugging ,
Flutter detected the device successfully and the app ran without any connection errors.
